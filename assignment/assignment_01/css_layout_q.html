기본 레이아웃 과제 01

  • 1. header/nav 는 상시 상단 고정으로 작업 할 것
  • 2. footer 는 상시 하단 고정으로 작업 할 것
  • 3. section 영역에서 스크롤이 발생할 것
  • 4. aside 영역에서 스크롤이 발생할 것
  • 5. section과 aside는 7:3 비율로 양옆으로 정렬이 될 것
  • 6. 영역 구분은 border나 background로 구분하여 주시기 바랍니다
